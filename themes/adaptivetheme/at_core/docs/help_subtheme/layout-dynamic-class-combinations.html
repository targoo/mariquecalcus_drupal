Dynamic Classes Cheat Sheet

Provided here are lists of the possible combinations of dynamic active-count and active-regions classes for two, three and four region combinations.

Two Regions

/* One active region */
.active-count-1 {}
.active-regions-1 {}
.active-regions-2 {}

/* Two active regions */
.active-count-2 {}
.active-regions-1-2 {}

Three Regions

/* One active region */
.active-count-1 {}
.active-region-1 {}
.active-regions-2 {}
.active-regions-3 {}

/* Two active regions */
.active-count-2 {}
.active-regions-1-2 {}
.active-regions-1-3 {}
.active-regions-2-3 {}

/* Three active regions use: */
.active-count-3 {}

Four Regions

/* One active region */
.active-count-1 {}
.active-regions-1 {}
.active-regions-2 {}
.active-regions-3 {}
.active-regions-4 {}

/* Two active regions */
.active-count-2 {}
.active-regions-1-2 {}
.active-regions-1-3 {}
.active-regions-1-4 {}
.active-regions-2-3 {}
.active-regions-2-4 {}

/* Three active regions: */
.active-count-3 {}
.active-regions-1-2-3 {}
.active-regions-1-2-4 {}
.active-regions-1-3-4 {}
.active-regions-2-3-4 {}

/* Four active regions use: */
.active-count-4 {}

Basic Usage Guide

Note, for more in depth information see the help section "CSS Selectors".

When using these classes for setting row region layout concantenate the "active-count" or "active-regions" class with the page row class. You will also need to target the actual regions inside the row to set layout. There are two ways of doing this - either by using the actual region names or by using psuedo selectors.

Contantenation with pseudo selectors:

.page-row-main.active-regions-1-2 .region:first-child {}
.page-row-main.active-regions-1-2 .region:last-child {}

Contantenation with actual region name selectors:

.page-row-main.active-regions-1-2 .region-highlighted {}
.page-row-main.active-regions-1-2 .region-content {}

You can cheat this when all regions are active and you want them all same width (a four column footer for example) e.g. here we use the .active-count class

.page-row-footer.active-count-4 .region {}

Another use of the active-count class is for when you have a two, three or four region layout and you just want to collapse the regions - e.g. if one is not being used just make the others wider:

.page-row-footer.active-count-2 .region {width: 50%}
.page-row-footer.active-count-3 .region {width: 33.3334%}
.page-row-footer.active-count-4 .region {width: 25%}

If you need more control over which regions get wider and maybe others do not, then you can use the active-regions class.
